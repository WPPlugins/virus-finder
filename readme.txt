== Description ==

Find viruses in your website with wphospital.hu. The plugin analyze all files, and shows the suspicious and virus files. 
After you can check it manually, and you can solve the problem!

== Installation ==

After you've downloaded and extracted the files:

1. Install Virus Finder automatically or by uploading the ZIP file. 
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on Virus Finder menu
4. Click "Scan Now"
